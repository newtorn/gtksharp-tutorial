## GTK

GTK＃是针对C＃编程语言的GTK +的包装。该库有助于使用Mono或任何其他兼容的CLR构建图形GNOME应用程序。使用Gtk＃构建的应用程序将在许多平台上运行，包括Linux，Microsoft Windows和Mac OSX。GTK＃是Mono计划的一部分。
